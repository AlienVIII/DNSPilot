# Windows Store Asset Slots

`Packaging/Package.Store.appxmanifest.template` expects these files before MSIX packaging:

- `StoreLogo.png`
- `Square44x44Logo.png`
- `Square150x150Logo.png`
- `Wide310x150Logo.png`
- `SplashScreen.png`

Keep the filenames stable or update the manifest template before generating the Store package.
