# Windows Store Assets

`Packaging/Package.Store.appxmanifest.template` expects these baseline PNG assets:

- `StoreLogo.png` - 50x50
- `Square44x44Logo.png` - 44x44
- `Square150x150Logo.png` - 150x150
- `Wide310x150Logo.png` - 310x150
- `SplashScreen.png` - 620x300

Keep the filenames stable or update the manifest template before generating the Store package.
Replace these only if final product branding changes.
