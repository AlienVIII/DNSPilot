# DNS Pilot Privacy Policy Draft

This draft matches the current Store-safe macOS build. Review it again before
publishing if telemetry, crash reporting, accounts, sync, remote catalog updates,
or a hosted backend are added.

## Data Collection

DNS Pilot does not collect personal data in the current Store-safe macOS build.

## Tracking

DNS Pilot does not track users across apps or websites.

## Analytics and Advertising

DNS Pilot does not include analytics, advertising SDKs, or third-party tracking
SDKs in the current Store-safe macOS build.

## Local Data

DNS Pilot stores the following data locally on the Mac:

- custom DNS profiles;
- custom domain test suites;
- benchmark history;
- app language and guided apply state.

This local data is used only to provide app features on the user's device.

## Network Activity

When the user runs a benchmark, DNS Pilot sends user-initiated DNS, TCP, and
optional connection-path probes to the selected DNS resolvers and target domains.
These requests are required for the benchmark feature.

## System DNS Changes

The Store-safe macOS build does not silently change system DNS. Guided apply
flows copy DNS servers and open macOS Network Settings so the user can make the
change directly.
