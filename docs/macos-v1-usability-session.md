# DNS Pilot macOS V1 Usability Session

Use this moderated script with five participants. Do not collect DNS query logs,
account details, or private resolver addresses.

## Setup

- Use the Store-safe build on the participant's normal network.
- Do not enable Direct Admin Actions or request an administrator password.
- Start with Check DNS selected and no prior explanation of the interface.

## Tasks

1. Run a quick DNS check and describe what happened.
2. Find a target for Dota 2 SEA, run it, and explain what the timing does not
   represent.
3. From a result, identify the recommended DNS and the safe next action.
4. Open the Apply confirmation and state whether DNS Pilot changed macOS DNS.
5. Find Profiles, save a sample plain DNS profile, and return to Check DNS.
6. Find Help/Setup and explain when Direct Admin Actions are available.

## Exit Criteria

- Five participants complete tasks 1, 3, and 4 without facilitator correction.
- No participant believes Store-safe Apply silently changes DNS.
- No participant describes game timing as in-match latency after task 2.
- Repeated confusion becomes a tracked product change before release.
